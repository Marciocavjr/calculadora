# 🎮 Calculadora Retro Game

Uma calculadora simples com visual de videogame clássico, feita com HTML, CSS e JavaScript.

## 💾 Como ele foi feito

- `index.html`: monta a estrutura da calculadora e os botões.
- `style.css`: deixa tudo com cara de retro game, cores escuras e botões estilo arcade.
- `script.js`: faz a mágica dos números, operações e do display.

A ideia foi criar uma calculadora básica que mostra o número digitado e calcula quando você aperta `=`.

## 🕹️ Como usar

1. Abra `index.html` no navegador.
2. Clique nos números para digitar um valor.
3. Clique em `+`, `-`, `x` ou `÷` para escolher a operação.
4. Digite outro número e aperte `=`.
5. `C` limpa a calculadora e volta para `0`.

## 🎯 O que funciona

- Digitar números
- Usar vírgula `,` como separador decimal
- Somar, subtrair, multiplicar e dividir
- Limpar a tela com `C`

## ⚡ Dica

Se quiser testar rápido, abra o arquivo `index.html` direto no navegador e divirta-se usando a calculadora com visual retrô!
